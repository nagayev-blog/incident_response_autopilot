Используй subagent **agent-builder** для создания нового агента с именем `$ARGUMENTS`.

Перед созданием:
1. Прочитай `CLAUDE.md` и `docs/architecture.md`, чтобы понять, в какое место графа агент впишется.
2. Прочитай хотя бы один существующий агент в `src/agents/` как эталон стиля. Если ни одного нет — используй принципы из системного промпта `agent-builder`.
3. Уточни у пользователя (если непонятно): какая роль агента, какие поля state читает, какие пишет, какая модель Claude (haiku для классификации / sonnet для рассуждений).

После создания файлов — напомни, что нужно подключить узел в граф через subagent `langgraph-architect`, и предложи это сделать.
